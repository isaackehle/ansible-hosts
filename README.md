# Ansible: setup-hosts

Update hosts file with a list of hosts

Available on Ansible Galaxy: [pgkehle.setup-hosts](https://galaxy.ansible.com/pgkehle/setup-hosts)

# Examples

```YAML
 - hosts: all
   roles:
     - pgkehle.setup-hosts
```
